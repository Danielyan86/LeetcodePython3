A non-empty array A consisting of N integers is given. The array contains an odd number of elements, and each element of the array can be paired with another element that has the same value, except for one element that is left unpaired.

For example, in array A such that:

A[0] = 9 A[1] = 3 A[2] = 9
A[3] = 3 A[4] = 9 A[5] = 7
A[6] = 9
the elements at indexes 0 and 2 have value 9,
the elements at indexes 1 and 3 have value 3,
the elements at indexes 4 and 6 have value 9,
the element at index 5 has value 7 and is unpaired.
Write a function:

def solution(A)

that, given an array A consisting of N integers fulfilling the above conditions, returns the value of the unpaired element.

For example, given array A such that:

A[0] = 9 A[1] = 3 A[2] = 9
A[3] = 3 A[4] = 9 A[5] = 7
A[6] = 9
the function should return 7, as explained in the example above.

Write an efficient algorithm for the following assumptions:

N is an odd integer within the range [1..1,000,000];
each element of array A is an integer within the range [1..1,000,000,000];
all but one of the values in A occur an even number of times.

### python 的位运算和 bool 运算

Python 中的 **位运算符** 是用来直接操作整数的二进制位的。以下是 Python 中常用的位运算符：

---

### **Python 位运算符**

| 运算符 | 描述            | 示例         | 结果                                  |
| ------ | --------------- | ------------ | ------------------------------------- | --- | ------------- | ------------------ |
| `&`    | 按位与（AND）   | `5 & 3`      | `1` （`0b0101 & 0b0011 = 0b0001`）    |
| `      | `               | 按位或（OR） | `5                                    | 3`  | `7` （`0b0101 | 0b0011 = 0b0111`） |
| `^`    | 按位异或（XOR） | `5 ^ 3`      | `6` （`0b0101 ^ 0b0011 = 0b0110`）    |
| `~`    | 按位取反（NOT） | `~5`         | `-6`（按位取反：`~0b0101 = -0b0110`） |
| `<<`   | 左移            | `5 << 1`     | `10`（`0b0101 << 1 = 0b1010`）        |
| `>>`   | 右移            | `5 >> 1`     | `2` （`0b0101 >> 1 = 0b0010`）        |

---

### **位运算符的详解**

#### **1. 按位与 `&`**

- 每一位都为 1 时，结果才为 1，否则为 0。
- **例子**：
  ```python
  a = 5  # 二进制：0101
  b = 3  # 二进制：0011
  print(a & b)  # 结果：1（0001）
  ```

#### **2. 按位或 `|`**

- 只要有一个位为 1，结果就为 1。
- **例子**：
  ```python
  a = 5  # 二进制：0101
  b = 3  # 二进制：0011
  print(a | b)  # 结果：7（0111）
  ```

#### **3. 按位异或 `^`**

- 两个位相同为 0，不同为 1。
- **例子**：
  ```python
  a = 5  # 二进制：0101
  b = 3  # 二进制：0011
  print(a ^ b)  # 结果：6（0110）
  ```

#### **4. 按位取反 `~`**

- 按位取反是对每一位取反，同时结果为 **负数**，按照**补码**表示。
- **例子**：
  ```python
  a = 5  # 二进制：0101
  print(~a)  # 结果：-6（二进制：~0101 = -0110）
  ```

#### **5. 左移 `<<`**

- 将二进制表示的数字向左移动若干位，相当于乘以 \( 2^n \)。
- **例子**：
  ```python
  a = 5  # 二进制：0101
  print(a << 1)  # 结果：10（二进制：1010）
  print(a << 2)  # 结果：20（二进制：10100）
  ```

#### **6. 右移 `>>`**

- 将二进制表示的数字向右移动若干位，相当于整除 \( 2^n \)。
- **例子**：
  ```python
  a = 5  # 二进制：0101
  print(a >> 1)  # 结果：2（二进制：0010）
  print(a >> 2)  # 结果：1（二进制：0001）
  ```

---

### **位运算的常见用途**

1. **快速计算**：

   - 左移和右移用于快速乘法或除法（适合 2 的幂的 �� 数）。

2. **掩码操作**：

   - 按位与 `&` 用于清除某些位 ���
   - 按位或 `|` 用于设置某些位。

3. **判断奇偶性**：

   - `n & 1`：结果为 0 表示偶数，结果为 1 表示奇数。

4. **交换两个变量（不使用临时变量）**：

   ```python
   a = 5
   b = 3
   a = a ^ b
   b = a ^ b
   a = a ^ b
   print(a, b)  # 输出：3, 5
   ```

5. **快速求集合的子集**（使用位掩码生成子集）。

---

### **注意事项**

1. 位运算符仅适用于**整数**。
2. 按位取反 `~` 会返回负数，因为 Python 使用补码表示整数。

如果需要更复杂的位运算应用，例如布尔运算，可以结合这些位运算符实现高效逻辑操作！- 位运算都是单个符号，&，

```python3
var = 0
for n in A:
    var = var ^ n

return var
```
